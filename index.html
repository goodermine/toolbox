      FAQ

      What metadata gets removed when cleaning?

      EXIF (camera, GPS, timestamps), XMP, IPTC/Photoshop blocks, JPEG comments, C2PA / "Content Credentials" / JUMBF provenance, and PNG/WebP text & metadata chunks — which is where tools like Stable Diffusion, DALL·E and Midjourney store generation parameters and "Made with AI" labels.

      How does the AI upsizer work?

      It runs an ESRGAN super-resolution neural network directly in your browser using TensorFlow.js. Unlike simple stretching, it reconstructs plausible detail for sharper edges and textures. The model files are bundled with this site (~3 MB total), so after the first load it works without any network connection. Fast mode uses classic high-quality canvas scaling for instant results on any device.

      Why can't I always upscale 4× on my phone?

      Phones (especially iPhones) limit how large an image a web page can hold in memory, so a very big 4× result can crash the tab. To prevent that, large jobs automatically switch to Fast mode, and anything beyond the device's limit is declined with a message instead of crashing. On a phone, 2× works for much larger images than 4× — and any computer will handle bigger results than a phone.

      Why is the first AI upscale slower?

      Because this tool is free and does everything on your device instead of on a paid cloud server. The first time you use AI mode, your browser downloads the model once (~3 MB) and "warms up" your graphics hardware — that's a few seconds of one-time setup. Everything after that runs from cache and is quick, even with no internet. If you need something instantly, Fast mode skips all of this.

      Will my image quality change?

      Cleaning is lossless by default — only metadata blocks are removed, pixels are untouched. Upsizing creates a new, larger image (saved as PNG to avoid any extra compression).

      Are my images uploaded anywhere?

      No. Open your browser's developer tools and watch the Network tab — after the page (and, for upsizing, the one-time model) loads, you'll see zero requests when you process an image.

      Can I host this myself?

      Yes — it's static files plus a small bundled model. Drop them on GitHub Pages, Netlify, Vercel, or any web host for free. See the README in the repository.
