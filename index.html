      How it works

      🔒

      100% private

      All processing happens locally in your browser. No image data is ever uploaded to a server — there is nothing to leak and nothing to limit.

      🧠

      Real AI, on your device

      Upsizing uses an ESRGAN super-resolution model that runs on your own hardware via TensorFlow.js — bundled with the site, so it works even offline.

      ♾️

      No daily limit

      Because the work runs on your own machine, you can clean and upsize as many images as you like — completely free.

      FAQ

      What metadata gets removed when cleaning?

      EXIF (camera, GPS, timestamps), XMP, IPTC/Photoshop blocks, JPEG comments, C2PA / "Content Credentials" / JUMBF provenance, and PNG/WebP text & metadata chunks — which is where tools like Stable Diffusion, DALL·E and Midjourney store generation parameters and "Made with AI" labels.

      How does the AI upsizer work?

      It runs an ESRGAN super-resolution neural network directly in your browser using TensorFlow.js. Unlike simple stretching, it reconstructs plausible detail for sharper edges and textures. The model files are bundled with this site (about 1 MB), so after the first load it works without any network connection. Fast mode uses classic high-quality canvas scaling for instant results on any device.

      Will my image quality change?

      Cleaning is lossless by default — only metadata blocks are removed, pixels are untouched. Upsizing creates a new, larger image (saved as PNG to avoid any extra compression).

      Are my images uploaded anywhere?

      No. Open your browser's developer tools and watch the Network tab — after the page (and, for upsizing, the one-time model) loads, you'll see zero requests when you process an image.

      Can I host this myself?

      Yes — it's static files plus a small bundled model. Drop them on GitHub Pages, Netlify, Vercel, or any web host for free. See the README in the repository.
