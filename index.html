    How it works

    🔒

    100% private

    All processing happens locally using JavaScript. No image data ever touches a server, so there is nothing to leak and nothing to limit.

    ✂️

    Lossless by default

    Metadata segments (EXIF / XMP / IPTC / C2PA / PNG text chunks) are surgically removed while your original pixel data is kept byte-for-byte.

    ♾️

    No daily limit

    Because the work runs on your own machine, you can clean as many images as you like, as fast as your computer allows.

    FAQ

    What metadata gets removed?

    EXIF (camera, GPS, timestamps), XMP, IPTC/Photoshop blocks, JPEG comments, C2PA / "Content Credentials" / JUMBF provenance, and PNG/WebP text & metadata chunks — which is where tools like Stable Diffusion, DALL·E and Midjourney store generation parameters and "Made with AI" labels.

    Will my image quality change?

    No, not in the default mode. We only delete metadata blocks and leave the compressed pixel data untouched, so the image is identical. The optional Deep clean mode re-draws the image to a canvas and re-encodes it, which guarantees removal of anything unusual but can slightly change the file.

    Are my images uploaded anywhere?

    No. Open your browser's developer tools and watch the Network tab — you'll see zero requests when you clean an image. You can even disconnect from the internet after the page loads and it still works.

    Can I host this myself?

    Yes — it's just three static files (HTML, CSS, JS). Drop them on GitHub Pages, Netlify, Vercel, or any web host for free. See the README in the repository.
